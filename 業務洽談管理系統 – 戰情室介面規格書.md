# 業務洽談管理系統 – 戰情室介面規格書

**版本：** v1.0  
**建立日期：** 2025-08-15  
**負責單位：** 川輝科技

## 1. 介面分區結構

整體畫面分為三個主要區塊：

### 左側 – 高優先案件列表

顯示經排序後的待追蹤案件（成交率高 → 下次追蹤日期近 → 預估金額大）

每個案件卡片包含：
- 成交率標籤（高 / 中 / 低，顏色區分：紅 / 黃 / 灰）
- 客戶名稱 + 專案名稱
- 預估金額
- 簽約日
- 下次追蹤日期（倒數顯示，例如「3 天後」）

支援篩選：成交率、高價案、來源、負責人

### 右上 – 案件詳情區

當選擇左側案件卡片時，右上顯示詳細資訊：

#### 基本資訊：
- 客戶名稱
- 專案名稱
- 成交率標籤
- 客戶預算狀態（符合 / 不夠 / 太低）
- 洽談階段
- 預估金額
- 付款條件

#### 分潤資訊：
- 分潤計算方式
- 分潤比例

#### 決策鏈資訊：
- 決策人姓名 / 職位 / 聯絡方式
- 關鍵影響者清單
- 決策權重（高 / 中 / 低）

#### 阻力與需求：
- 主要痛點（多選）
- 成交阻力標籤
- 競爭對手名稱與狀態

### 右下 – 行動追蹤紀錄區

顯示該案件的所有行動紀錄（倒序排列，最新在上）

每筆紀錄包含：
- 時間戳記（YYYY/MM/DD HH:mm）
- 行動類型（電話 / 面談 / 簡報 / 報價 / 寄資料 / 送樣）
- 行動內容摘要

#### 新增行動按鈕：
- 選擇行動類型
- 輸入行動內容
- 設定下次追蹤日期（必填）
- 附件上傳（選填，支援 PDF / JPG / PNG / DOCX / XLSX）

## 2. 功能互動流程

### 案件優先排序邏輯（系統自動）

#### 主要排序條件：
1. 成交率標籤：高 > 中 > 低
2. 下次追蹤日期：近 > 遠
3. 預估金額：大 > 小

若同條件則依最後更新時間排序（最新在前）

### 提醒機制
- 下次追蹤日期到 → 系統通知（Email / APP / LINE OA）
- 簽約日逼近（7 天內）→ 提醒跟進
- 超過 14 天未更新 → 黃牌提示
- 高價案（>50萬）且成交率高 → 標註「優先」

### 案件更新
每次業務更新「目前狀態說明」與「行動紀錄」時，系統自動產生時間戳記

更新時可同時：
- 上傳檔案
- 調整成交率標籤
- 設定下次追蹤日期

### 篩選與搜尋
支援多條件篩選：
- 成交率（高/中/低）
- 預算狀態
- 負責人
- 洽談階段
- 客戶來源

## 3. 欄位資料規格

我已在之前的 PRD 中列出欄位清單，這裡是前端顯示用的簡化表：

### 案件卡片顯示欄位（左側列表）

| 欄位 | 顯示方式 |
|------|----------|
| 成交率標籤 | 圓角色塊（紅/黃/灰） |
| 客戶名稱 | 粗體 |
| 專案名稱 | 常規字 |
| 預估金額 | 右側顯示，千分位 |
| 簽約日 | 日期 + 「剩餘天數」 |
| 下次追蹤日期 | 日期 + 倒數 |

### 案件詳情（右上）
- **基本資訊區塊：** 表格呈現
- **決策鏈區塊：** 列表呈現（可折疊）
- **阻力與需求區塊：** 標籤樣式（可多選）

### 行動紀錄（右下）
Timeline 時軸樣式

每筆顯示：
- 日期時間
- 行動類型標籤
- 行動內容
- 附件圖示（如有）

## 4. 權限

- **業務：** 新增/編輯自己案件，查看自己案件列表
- **主管：** 查看所有案件，可編輯屬下案件
- **財務：** 僅查看財務欄位與報表
- **老闆：** 全權

## 5. 響應式設計

- **桌面版：** 三欄布局（左側列表 / 右上詳情 / 右下行動）
- **平板：** 兩欄（列表 / 詳情 + 行動切換）
- **手機：** 單欄（列表 → 點擊進入詳情頁 → 再切到行動紀錄頁）
